html 첫 수업

html이란?

Hyper Text Markup Language의 약자.
웹문서를 만드는 표준 언어.
구조를 담당.

태그(명령어)로 구성 됨.

☁html문서의 필수 태그들

   <html>     ----------- html문서의 시작
    <head>      ----------- 머리말의 시작
     <title>     </title>  ----- 제목
    </head>     ----------- 머리말의 끝 
    <body>      ----------- 본문의 시작

    </body>     ----------- 본문의 끝
   </html>    ----------- html문서의 끝

☁hn(n=1~6) tag - 제목 글자 태그

bold체, 위아래 한 줄 공백, 글자 크기 변화

body(본문)에 들어가는 태그들

hn = 제목 글자
p = 문단(위아래 한 줄 공백)
br = 줄바꿈
hr = 수평선
pre = 소스 그대로 공백 유지(고정폭글꼴)

(예약어 포함) 특수문자태그들

- &로 시작하고 ;로 끝남 -
(tag) = (결과)
&lt; = <
&gt; = >
&amp; = &
&nbsp; = 공 백
(ex) coffee&tea, black&white

&copy; = ©
&reg; = ®
&trade = ™

글자 모양과 관련된 태그들

b = bold체
i = 이텔릭체
small = 글자크기 작게
sup = 윗첨자 (ex) 2cm2
sub = 아랫첨자 (ex) H2O
