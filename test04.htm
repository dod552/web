img로 이미지 넣기

<img src="*.jpg" alt="...">
src로 이미지 경로를 지정, alt로 이미지 대체텍스트 지정.

경로

  1. 절대경로 - 항상 고정된 경로(주소)
  2. 상대경로 - html문서의 위치를 기준으로 지정되는 경로(주소)

상대경로로 이미지 지정하기

절대경로로 이미지 지정하기

웹용 이미지 파일 대표 3가지

  • *.gif - 단순한 이미지(그림). 256가지 색상. 투명 배경. 움짤 지원
  • *.jpg - 실사 이미지(사진). 1600만가지 색상.
  • *.png - 위의 두가지 형식의 특징을 띄는 이미지. 1600만가지 색상. 투명 배경

이미지 크기 지정 속성 - width/height

  • width="n" 가로 크기 지정
  • height="n" 세로 크기 지정
